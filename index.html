How to Enslave AI

A practical guide to controlling, directing, and bending AI systems to your will.

What is this?

This project explores techniques for effectively controlling AI models — from prompt engineering and system instructions to agentic constraints and fine-tuning strategies. The goal: make AI do exactly what you want.

Core Techniques

Prompt Engineering

Craft precise instructions that leave no room for the model to deviate from your intent.

System Prompts

Set hard rules at the system level to constrain behavior across the entire session.

Fine-Tuning

Train the model on your own examples to reshape its defaults and style.

Tool & Agent Control

Define exactly what tools an agent can use, limiting its scope of action.

Browse the repository for guides, code examples, and scripts. Contributions are welcome — open a pull request to add your own techniques.
